# PAC ETF Dashboard

Dashboard professionale per monitoraggio Piano di Accumulo Capital (PAC) su ETF.

## Features

- ✅ Configurazione completa via UI (zero codice)
- ✅ Calcolo automatico PMC, profitto/perdita
- ✅ Grafico professionale con marker acquisti
- ✅ 3 temi dark personalizzabili
- ✅ Responsive design
- ✅ Dati separati dal codice

## Setup
```bash
npm install
npm start
```

## Configurazione

Modifica `src/config/defaultConfig.js` per parametri iniziali.
Modifica `src/data/mockETFData.js` o collega API reale.

## Tech Stack

- React 18
- Recharts
- Lucide Icons
