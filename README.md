# PAC Dashboard Pro (Static Version)

Dashboard stile TradingView per il monitoraggio di PAC su ETF.

## Caratteristiche
- **No Installazione**: Basta aprire `index.html`.
- **TradingView Charts**: Grafici interattivi professionali.
- **Layout Responsive**: Griglia CSS avanzata stile Bloomberg.
- **Dati Mock**: Generazione automatica di dati per demo.
- **Configurabile**: Modifica importo mensile e giorno d'acquisto.

## Come usare
1. Apri `index.html` nel tuo browser.
2. Clicca l'icona ingranaggio (⚙️) per configurare il tuo PAC.

## Deploy su GitHub Pages
Basta caricare questi file nella root del repository e attivare GitHub Pages dalle impostazioni.
